# Internship Project - Course Platform UI

## 📌 Overview

This project is a simple frontend web application built as part of an internship task. It represents a course platform landing page inspired by modern ed-tech websites.

The application showcases course offerings, career-focused content, and a user form to help visitors find suitable courses.

## 🚀 Live Demo

Deployed on Vercel:
https://internship-109.vercel.app/

## 🛠️ Tech Stack

* Next.js (App Router)
* React
* Tailwind CSS
* TypeScript

## ✨ Features

* Responsive landing page design
* Reusable components (Navbar, Hero Section, Offerings Section)
* Course offerings display (Bootcamps, Certifications, New Launches)
* Statistics section highlighting achievements
* User input form (experience, interest, contact details)
* Clean UI with modern design (Tailwind CSS)

## 📂 Project Structure

* `components/`

  * Navbar
  * HeroSection
  * OfferingsSection
* `app/`

  * Main page layout and sections
* Reusable UI sections for scalability

Example:

* Navbar, Hero, and Offerings are imported and used in the main page 

## 📸 Sections Included

* Navigation Bar
* Hero Section
* Stats Section
* Course Finder Form
* Course Offerings
* Success Metrics
* Features / Advantages
* Footer

## 🎯 Purpose

The main goal of this project was to:

* Practice component-based architecture
* Build a clean and responsive UI
* Understand real-world landing page structure
* Deploy a Next.js application using Vercel

## ⚙️ Installation & Setup

```bash
# Clone the repository
git clone <your-repo-link>

# Navigate into project
cd <project-folder>

# Install dependencies
npm install

# Run development server
npm run dev
```
